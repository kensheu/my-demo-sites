## 功能需求清單

> 各頁面皆須附有填寫指南，說明欄位規範與注意事項。

### 首頁公告

- 顯示位置：
    - 內部後台首頁
    - 訂餐前台首頁
    - 餐廳後台首頁
- 公告內容包含：
    - 文字、圖片、附件
    - 公告分類：更新通知、維護公告、活動訊息
- 通知功能（三個位置皆支援）：
    - 自動跳出未讀公告通知
    - 查看全部公告，可依全部 / 未讀 / 公告分類篩選
    - 標記已讀
- 公告管理頁面（僅內部後台）：
    - 新增公告，包含：標題、分類（下拉選擇）、顯示位置（複選）、內容、圖片、附件
    - 可設定排程（發布時間、到期時間）
    - 公告狀態：發布中、草稿、排程待發、已到期
    - 搜尋標題、依分類 / 狀態 / 顯示位置篩選
    - 編輯、刪除公告

### 客訴管理

- 使用對象：
    - 會員前台（提出客訴）
    - 管理後台（處理客訴）
- 會員前台功能：
    - 從我的訂單發起客訴，自動帶入訂單編號、據點、客戶資訊
    - 選擇客訴類型、描述問題、上傳照片
    - 查看我的客訴清單與狀態
- 管理後台功能：
    - 收到新客訴時需有通知提醒
    - 客訴清單篩選（據點、類型、狀態、日期）
    - 處理進度步驟條
    - 補償方式：折價券或退款
    - 處理紀錄新增與查看
    - 後台回覆後，會員前台需收到通知

### 問題單與 ECR 請求

- 通知機制：
    - 新問題單提交後，IT 工程師需收到通知
    - 問題單狀態更新或回填後，填寫人需收到通知
    - 新 ECR 提交後，主管需收到審核通知
    - ECR 審核結果出爐後，填寫人需收到通知
- 範本功能（可行的話）：
    - 問題單與 ECR 皆支援儲存為範本，下次可快速套用常用欄位

- 問題單 — 填寫區：
    - 必填欄位：
        - 部門、填寫人
        - 專案名稱、模組
        - 問題摘要
        - 嚴重程度（嚴重 / 高 / 中 / 低）
        - 問題類型（程式錯誤、資料錯誤、畫面異常、效能問題、版面問題、規格異動）
        - 哪一支網頁 / 程式、發生時間
        - 進行了什麼操作（操作步驟）
        - 發生了什麼問題（實際結果 vs 預期結果）
        - 以前有沒有發生過？
        - 其他人有沒有發生過？
        - 作業系統、瀏覽器
    - 選填欄位：裝置型號、正常環境資訊、重現步驟、測試資料、截圖 / 錄影附件
- 問題單 — 清單區：
    - 搜尋、嚴重度、狀態篩選
    - 統計數量顯示（全部、待處理、處理中、已解決）
    - 編輯、刪除問題單
    - IT 工程師回填：問題原因、修復方式
- ECR — 填寫區：
    - 必填欄位：
        - 專案名稱、模組
        - 申請日期、部門、填寫人
        - 變更標題
        - 變更類型
        - 優先級
        - 原因
        - 現行狀況（AS-IS）
        - 變更後期望狀況（TO-BE）
        - 影響的頁面 / 程式
        - 影響程度
        - 期望完成日期
    - 選填欄位：相關問題單編號、相依性、附件、備註
    - 主管進行審核（核准 / 退回 / 需討論）
- ECR — 清單區：
    - 搜尋、優先級、狀態篩選
    - 統計數量顯示（待審核、已核准、實施中）
    - ECR 狀態：草稿、待審核、審核中、已核准、已退回、實施中、已完成
    - 編輯、刪除 ECR
    - 主管回填審核結果（核准 / 退回 / 需討論）與審核意見

### 餐點價格異動紀錄

- 共用操作：
    - 店家切換
    - 月份切換（上 / 下一月）
- 價格快照頁：
    - 顯示當月全部品項完整價格清單
    - 欄位：料號、餐點名稱、成本價、店內價、終端售價、異動標記
    - 異動標記：新增、異動、已下架、無異動
    - 統計數量（共幾筆、各標記數量）
- 異動事件紀錄頁（流水帳）：
    - 依日期分組呈現每筆異動，顯示異動前後的值
    - 每筆紀錄包含：料號、餐點名稱、異動描述、日期、時間、操作人員、來源（人工操作 / 系統自動）
    - 依異動類型篩選（成本 / 售價異動、品名異動、上下架 / 新增）
    - 搜尋品名或料號
